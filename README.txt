Steps to run eye blinking user authentication system:

1. Open directory from terminal, ensure,you are running python3 with dlib, cv2,and imutils.

2. To execute type
   $: python main.py
   Follow the instructions on the terminal.

3. Tips:
 Make sure username is atleast 4 characters long.

 Be patient while calibration.

 You can reset the password while either entering or authenticating by pressing the 'r' key.
 Read the pdf attached to understand the working of the algorithm.
